# AdNetwork-w-Splash

- Pulling from scrapinghub/splash
  
  ```
  docker pull scrapinghub/splash
  ```

- Run docker and visit Splash on port 8050(http)
  
  ```
  docker run -p 8050:8050 scrapinghub/splash
  ```
